# cerulio-bootstrapper
Bootstrapping core open data for local use

| Dataset name                         | License                                   | Description |
|--------------------------------------|--------------------------------------------|-------------|
| Overture Maps                        | Permissive open licenses (e.g., CDLA)       | Global, open vector map data including roads, buildings, places, and administrative features with permissive reuse terms. |
| Fieldmaps Global Boundaries          | Open Data Commons Open Database License (ODbL) or Public Domain * | Edge-matched global administrative boundaries (international and subnational) assembled for humanitarian operational reference, combining UN CODs, OSM, and other authoritative sources. :contentReference[oaicite:0]{index=0} |
| GHSL Population Density              | CC BY 4.0                                  | High-resolution global population density grids produced by the European Commission, useful for urban analysis. |
| Copernicus DEM                       | Free under Copernicus/ESA terms             | Global digital elevation model; free access with attribution and user terms. |
| GLC_FCS10 Global Land Cover          | CC BY 4.0                                  | Updated 10 m global land cover classification with refined classes for urban, agriculture, forest, and water. |
